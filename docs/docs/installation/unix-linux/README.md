# Installation on Unix/Linux <seo/>

## Requirements 

- Build Essential - (build-essential package)
- Git
- CMake 

## Build

```bash
$ git clone https://github.com/oatpp/oatpp.git
$ cd oatpp/

$ mkdir build && cd build

$ cmake ..
$ make install
```
